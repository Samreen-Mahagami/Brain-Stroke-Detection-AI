## ğŸš€ Quick Start

### 1. Deploy Phase 1

```bash
cd phase1_ingestion
./deploy.sh
```

### 2. Upload DICOM File

```bash
# Upload to S3
aws s3 cp your_scan.dcm s3://stroke-detection-ai-dicom-uploads-{ACCOUNT_ID}/studies/STUDY-001/

# Trigger processing
curl -X POST https://{API_ID}.execute-api.us-east-1.amazonaws.com/prod/upload \
  -H 'Content-Type: application/json' \
  -d '{
    "patient_id": "P123456",
    "file_key": "studies/STUDY-001/your_scan.dcm",
    "study_description": "Brain CT - Stroke Protocol"
  }'
```

### 3. Check Status

```bash
# Query DynamoDB
aws dynamodb scan --table-name stroke-detection-ai-study-metadata

# Check Step Functions
aws stepfunctions list-executions \
  --state-machine-arn arn:aws:states:us-east-1:{ACCOUNT}:stateMachine:stroke-detection-ai-processing
```
